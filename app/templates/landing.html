Comprehensive Features

Everything you need to manage biodiversity collections professionally

Collection Management

Organize and catalog specimens with full support for Darwin Core fields, taxonomic data, and geographic information.

Spatial Data Support

GeoAlchemy2 integration for geographic data management with PostgreSQL spatial extensions.

Multi-Language i18n

Built-in internationalization with Flask-Babel supporting Traditional Chinese and English interfaces.

Multi-Site Architecture

Support multiple institutions on a single platform with site-specific configurations and templates.

Role-Based Access

Secure authentication with Flask-Login and JWT, supporting admin, curator, and researcher roles.

RESTful API

Comprehensive API endpoints for data integration, automation, and third-party application development.

Get Started

Deploy your own instance in minutes

1

Clone Repository

git clone https://github.com/TaiBIF/naturedb.git
2

Configure Environment

cp dotenv.sample .env

Edit .env with your database credentials and settings

3

Build & Run with Docker

docker compose -f compose.yml -f compose.override.yml up
4

Access Your Instance

Open http://localhost:5000 in your browser
